pageTitle: Utility Tokens
---

```scss
'align': (
  'items': (
    'left': 'left',
    'center': 'center',
    'right': 'right',
  ),
  'property': 'text-align',
),
```

Will generate the following css

```css
.u-align-left {
  text-align: left;
}

.u-align-center {
  text-align: center;
}

.u-align-right {
  text-align: right;
}
```
